
1. Make code changes in Visual Studio

2. Update AssemblyInfo.cs version number (e.g. 2.2.2.17)

3. Build in Release mode (Ctrl+Shift+B) — confirm 0 errors

4. Git Changes panel — stage all, commit message, Commit All, Push

5. Terminal — tag and push:
      git tag v2.2.2.17
      git push origin v2.2.2.17

6. Update version.json in the public GitHub repo:
      - version: "2.2.2.17"
      - downloadUrl: update to new version number
      - releaseNotes: describe what changed

## GitHub Actions builds and publishes the zip automatically after step 5.
## No manual zip or upload needed.
